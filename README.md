## MAF key combinations:
RC - Stop       = Shift + Down.
RC - Play/Pause = Shift + Up.
RC - Rewind     = Shift + Left.
RC - Forward    = Shift + Right.
RC - Back       = Backspace.
Red             = F1.
Green           = F2.
Yellow          = F3.
Blue            = F4.
Channel Up      = PageUp.
Channel Down    = PageDown.
